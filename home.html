Ideation

My project is based around the ideas of computer programming and visualizing this an interactive website. This website aims to help everyday people understand how computer programming works. People don’t realize how much programming can go into creating a single webpage on a website.

This idea will be displayed through a website, so all users will be able to navigate around. It will show the user what programming is, what it means to be a programmer, the languages used, and also teach the users how to programming themself.

The idea behind the website is that there will be four pages

  • Home page - Necessary information introducing the topic
  • What is programming - What is programming, What is a programmer, languages
  • Become a programmer - Links to another code-based website, Learners guide to programming
  • News/events - Facts about programming achievements, Events on programming

Original Idea

The first design is a straightforward layout choice. The reason I went for the simple layout wast to make sure those of all ages would be able to navigate around each of the pages of the website. I have the use of the navy blue as the background to bring colour into the web design, so it wasn’t just white. On each of the four pages, the layout is similar to show the links between the pages of the website.

All the information for each of the pages is all included to make the layout easier. It also improves the usability of the design, so all ages will then find the information they need. Though the design is good for the initial idea, I wonder if it is a bit dull in the colour used. I think I might play around with the colours, and see if I can then make it more contemporary and pleasing to the eye

Prototyping

After creating the original idea, I started to play around with the layout and the colours to see if I could make it more pleasing to the eye. One of the main aspects I changed was the background colour. I took out the navy blue and made it white. With this change in the design, the website becomes fresher and more of a contemporary website. I also changed the colour of the nav background, making it white to match the background.

Some of the other aspects I then changed where the layout of the information on each page. I took away the sections from the website, as it made it look too busy, and added in a header. Each of the headers is an image to help bring colour to the website. The home page is a bit different from the original idea, as I have now made it into a landing page instead, which contains three buttons that link to the other pages of the website.

Icons

The icons used are simple, but they help give the areas extra meaning to them. The use of icons helps to fit for those who have trouble reading, making it easier for them to follow along. At first, I wasn’t going to add the icons as I couldn’t see how it would fit, but after testing, it helped improve the layout.

Four main icons are being used on this website. Three are being used to help represent the programming information. The fourth one is being used in three different ways. It is being used to represent the level in which the user wants to code. This is to make sure the user is going to practice code at a level that is wrong for them.

Development

Pop-up Page

To add extra detail to the website and make it more interactive, I have chosen to add on a pop-up page. At first, the initial idea was to have the user scroll down the page. Though I found design-wise, it could just make the page look too busy, so I decided to change it up a bit. By adding on the pop-up page, it makes it a more interactive website. How I have made it work is by having three big buttons that have an icon and a title of the information which will follow. It always helps with keeping the webpage clean and tidy so that it won’t be overcrowded.

Links

For my next two pages, they are based more on the idea of links. For the 'how to become a programmer' page, there are three buttons – beginner, medium, and pro. These three buttons are represented by icons, as mentioned before. When clicked, it will take you to a related website where you can learn to program at the level of the user currently is. The website I have used is code avengers and code academy.

The news and event page, like the other, is similar. I have designed the website into two categories, one for news and the other events. I choose to do this page cause I hadn't realized how many accomplishments there have been on programmers, and more people should be aware of it. There are also so many different programming events that happen worldwide via the internet. This shows how programming can bring everyone together, no matter where you are. Just like the other page, this page has links to another website where the info is held. Though on each of the buttons is the name of the news article or the name of the event.

Because this is just a prototype of my website, I could not figure out on Figma to make actual links to websites.
